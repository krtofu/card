## 시작하기

· 개발 서버 실행 코드

```bash (터미널에 입력!)
npm run dev 
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
· 개발 서버 주소 ⋆ ⋆ ⋆ ⋆ ⋆
[Sekard / 세카이 두부도감](http://localhost:3000)

· 페이지 수정하기 (기본)

`app/page.tsx`파일을 수정하면 페이지를 바로 편집할 수 있습니다. 파일을 저장(CTAL+S) 하면 변경 사항이 자동으로 반영됩니다.

· 폰트 최적화

이 프로젝트는 [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts)를 사용해 Vercel의 새로운 폰트 패밀리인 [Geist](https://vercel.com/font)를 자동으로 최적화하고 로드합니다.
